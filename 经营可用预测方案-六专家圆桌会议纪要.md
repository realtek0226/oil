# 经营可用预测方案-六专家圆桌会议纪要

## 会议结论

当前系统不能把预测点位直接作为经营依据。经营可用版本应改为：

```text
参考中枢 + 历史误差区间 + 经营可用性等级 + 事件风控闸门 + 可执行区域价差
```

点位只作为参考中枢，真正触发采购、销售、库存、跨区发运动作的是经营可用性等级和触发条件。

## 六位专家共识

1. 山东地炼现货专家：方向不再只展示上涨/下跌，改为明确上涨、震荡偏强、震荡、震荡偏弱、明确下跌。预测变化小于历史RMSE一半时，不允许给强单边结论。
2. 量化校准专家：高RMSE下点位不能独立指导经营。只有预测变化达到0.8倍RMSE、方向概率足够高、可靠度不低时，才允许明确上涨或明确下跌。
3. 贸易库存专家：经营建议必须写成“建议动作、触发数据、执行幅度、止损条件”，不能只写建议关注。
4. 发改委政策专家：调价窗口必须作为硬约束，窗口临近时自动降低跨窗口囤货和重仓动作。
5. 区域套利专家：区域价差必须升级为可执行价差，计算口径为“目标区域92#-山东92#-运费-损耗-装卸-资金成本-信用缓冲”。
6. 原油事件专家：黑天鹅、地缘、OPEC、制裁等事件先进入风控闸门。高风险事件下不展示强单边经营结论，极高风险切人工决策。

## 已落地修改

1. 山东92#预测增加经营可用性分层：
   - A/B：可作为经营动作触发依据。
   - C：只能作盘中观察或轻动作，不允许放量、重仓补库、大幅让利。
   - D：不作为自动经营依据，必须人工复核。

2. 增加RMSE约束：
   - `预测变化 / RMSE < 0.5`：展示为震荡、震荡偏强或震荡偏弱。
   - `预测变化 / RMSE >= 0.8` 且方向概率、可靠度达标：才允许明确上涨或明确下跌。

3. 增加事件风控闸门：
   - 低：模型正常展示。
   - 中：保留预测，但经营建议降为轻动作。
   - 高：隐藏强单边措辞，人工确认后再执行。
   - 极高：暂停自动经营结论，切换人工决策。

4. 区域价差改为三层口径：
   - 区域价差 = 目标区域92#价格 - 山东92#价格。
   - 净回款价差 = 区域价差 - 运费。
   - 可执行价差 = 净回款价差 - 损耗 - 装卸 - 资金成本 - 信用缓冲。

5. 跨区发运动作改为阈值触发：
   - 可执行价差 > 60元/吨：连续两次为正后扩大外发。
   - 30-60元/吨：小批量试发。
   - 0-30元/吨：只做客户维护或锁价订单。
   - <0元/吨：原则停发。

6. 前端展示改造：
   - 点位改为“参考中枢”。
   - 区间改为“经营参考区间 + 历史误差区间”。
   - 首页新增经营可用性、事件风控、可执行价差、经营动作标签。

## 当前样例

2026-06-01 本地测试结果：

```text
D1预测变化：-21.47元/吨
RMSE：140.67元/吨
预测变化/RMSE：0.15
展示口径：震荡
经营等级：C
经营含义：可作盘中观察，不允许直接触发放量、重仓补库或大幅让利
```

这个结果说明：即使模型点位低于现货，也不能给“明确下跌”经营结论，因为预测变化远小于历史误差。
