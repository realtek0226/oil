# 成品油询价竞厂价格查询接口文档

- 接口版本：v1
- 更新日期：2026-06-15
- 维护方：定价系统（djgl）

## 接口信息

| 内容 | 信息 |
|---|---|
| 接口中文名称 | 成品油询价竞厂价格查询接口 |
| 接口用途、功能 | 查询成品油询价中，产品 92#VI、0#VI 在指定日期下、除京博外所有竞争对手的昨日价格与今日价格 |
| 调用系统名称 | 定价系统（djgl） |
| 接口类型 | RESTFUL |
| URL | http://10.189.2.50:8087/price/openapi/competitorPrice |
| 请求方法 | GET |
| 请求PATH |  |
| 请求QueryString | day |
| 请求header |  |
| 请求body | GET 请求无 body |
| 请求参数说明 | {"day": "查询日期，格式 yyyy-MM-dd，不传时默认当天"} |
| 外网访问需求 | 无（内网/网络白名单访问，本期暂不鉴权） |
| 调用成功样例 | {"code": 0,"msg": "success","data": {"day": "2026-06-15","products": [{"product": "92#VI","items": [{"company": "东明石化","yesterdayPrice": 7800,"todayPrice": 7850}]},{"product": "0#VI","items": [{"company": "东明石化","yesterdayPrice": 7000,"todayPrice": 7050}]}]}} |
| 返回参数说明 | {"code": "状态码，0成功非0失败","msg": "状态描述","data": {"day": "实际查询日期","products": [{"product": "产品名（92#VI/0#VI）","items": [{"company": "竞争对手名称","yesterdayPrice": "昨日价格（元/吨，未录入为null）","todayPrice": "今日价格（元/吨，未录入为null）"}]}]}} |
| 调用失败样例 | {"code": 1,"msg": "日期格式错误，需为 yyyy-MM-dd","data": null} |

## 请求

### 请求示例

```bash
curl "http://10.189.2.50:8087/price/openapi/competitorPrice?day=2026-06-15"
```

不传日期（默认当天）：

```bash
curl "http://10.189.2.50:8087/price/openapi/competitorPrice"
```

## 响应

### 调用成功样例

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "day": "2026-06-15",
    "products": [
      {
        "product": "92#VI",
        "items": [
          { "company": "东明石化", "yesterdayPrice": 7800, "todayPrice": 7850 },
          { "company": "鲁清石化", "yesterdayPrice": 7750, "todayPrice": 7800 }
        ]
      },
      {
        "product": "0#VI",
        "items": [
          { "company": "东明石化", "yesterdayPrice": 7000, "todayPrice": 7050 }
        ]
      }
    ]
  }
}
```

### 返回参数说明

| 字段 | 类型 | 说明 |
|---|---|---|
| `code` | int | 状态码，`0` 表示成功，非 `0` 表示失败 |
| `msg` | string | 状态描述 |
| `data` | object/null | 成功时为数据对象，失败时为 `null` |
| `data.day` | string | 实际查询日期（`yyyy-MM-dd`） |
| `data.products` | array | 产品列表，固定顺序：`92#VI` 在前，`0#VI` 在后 |
| `data.products[].product` | string | 产品名（`92#VI` / `0#VI`） |
| `data.products[].items` | array | 该产品各竞厂价格；无记录时为空数组 `[]` |
| `data.products[].items[].company` | string | 竞争对手名称（不含京博） |
| `data.products[].items[].yesterdayPrice` | number/null | 昨日价格（元/吨）；未录入时为 `null` |
| `data.products[].items[].todayPrice` | number/null | 今日价格（元/吨）；未录入时为 `null` |

### 调用失败样例

日期格式错误：

```json
{ "code": 1, "msg": "日期格式错误，需为 yyyy-MM-dd", "data": null }
```

服务异常：

```json
{ "code": 2, "msg": "服务异常，请稍后重试", "data": null }
```

## 错误码

| code | msg | 触发条件 |
|---|---|---|
| `0` | `success` | 查询成功（含无数据的空结果） |
| `1` | `日期格式错误，需为 yyyy-MM-dd` | `day` 参数格式不合法（如 `2026/6/15`） |
| `2` | `服务异常，请稍后重试` | 数据库查询等内部异常 |

## 数据说明

- **价格单位**：元/吨。
- **"今日/昨日"含义**：取询价表中每条记录的 `price`（当日报价）与 `yesterday_price`（前一日报价），由询价录入流程维护。
- **价格可能为空**：竞厂当日尚未报价时，对应字段为 `null`，并非错误。
- **竞争对手范围**：结果中不含京博自身，仅返回竞厂。
- **数据时效**：当日数据取决于询价录入进度；建议在业务录入完成后查询。
