# 数据采集脚手架说明

新增脚本：[scripts/refined_oil_collectors.py](</E:/中鲁燃能/scripts/refined_oil_collectors.py>)

## 1. 已落地能力

### 1.1 发改委最新调价公告

命令：

```powershell
python scripts/refined_oil_collectors.py ndrc-latest
```

默认输出：

`artifacts/ndrc_latest_notice.json`

当前会自动：

- 从国家发改委新闻发布列表页发现最新成品油公告
- 进入详情页抽取：
  - 标题
  - 发布日期
  - 发布部门
  - 汽油/柴油每吨涨跌幅
  - 生效时间

### 1.2 省级发改委样例页

配置文件：

[configs/province_price_templates.yaml](</E:/中鲁燃能/configs/province_price_templates.yaml>)

命令：

```powershell
python scripts/refined_oil_collectors.py local-fgw-samples
```

默认输出：

`artifacts/local_fgw_samples.json`

当前已接两种模板：

- 上海：`标号 + 元/吨 + 元/升` 双行结构
- 浙江：`零售价 + 批发价` 单行结构

当前已预留但未启用：

- 山东
- 江苏
- 广东

### 1.3 易加油开放平台接入骨架

文档样例归一化：

```powershell
python scripts/refined_oil_collectors.py ejiayou-sample
```

默认输出：

`artifacts/ejiayou_station_pages_sample_normalized.json`

实际接口调用：

```powershell
$env:EJIAYOU_BASE_URL='https://pre.ejiayou.com'
$env:EJIAYOU_PLATFORM_NAME='your_platform'
$env:EJIAYOU_BEFORE_KEY='your_before_key'
$env:EJIAYOU_AFTER_KEY='your_after_key'
python scripts/refined_oil_collectors.py ejiayou-stations --page 1
```

默认输出：

`artifacts/ejiayou_station_page.json`

当前实现了：

- `sign = MD5(beforeKey + timestamp + afterKey).upper()`
- 油站分页接口 URL 生成
- 油站价格结果归一化

默认分页路径模板可通过环境变量覆盖：

`EJIAYOU_STATION_PAGES_PATH_TEMPLATE`

默认值：

```text
/oreo/ejiayou_open_api/stations/v2/getStationPages/{currentPage}/{pageSize}/{platformName}/{sign}/{timestamp}
```

示例环境变量文件：

[configs/ejiayou_api.example.env](</E:/中鲁燃能/configs/ejiayou_api.example.env>)
